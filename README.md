Responsive Landing Page with Interactive Navigation
Task Overview

This project demonstrates a responsive landing page with a fixed navigation menu that dynamically changes its style on scroll and hover, enhancing user interaction and usability.

 Features

Fixed navigation bar visible on all sections

Navigation style changes on page scroll

Hover effects on menu items

Fully responsive design for desktop and mobile devices

Clean and simple UI using HTML, CSS, and JavaScript

 Technologies Used

HTML5 – Structure of the web page

CSS3 – Styling, responsiveness, and animations

JavaScript – Scroll-based interaction logic

 Project Structure
├── index.html
├── style.css
├── script.js
└── README.md

 How to Run the Project

Download or clone the repository

Open index.html in any modern web browser

Scroll the page and hover over the navigation items to see the effects

Learning Outcomes

Understanding fixed navigation menus

Implementing scroll-based UI interactions

Creating responsive layouts

Enhancing user experience with hover effects
